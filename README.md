# 拾元文件夹

纯本地运行的文件夹管理器，一个 HTML 文件搞定。仿 Apple 设计风格，支持多文件夹管理、常见文件预览、Markdown 和 HTML 编辑保存。

## 使用方式

双击 `index.html` 即可在浏览器中打开，无需安装任何软件。

## 功能

### 文件夹管理
- **多文件夹同时管理** — 侧边栏可添加多个文件夹，各自独立展开
- **拖拽添加** — 从访达 `⌘+点击` 多选文件夹，拖入侧边栏一键添加
- **新建文件/文件夹** — 右键或点击工具栏按钮创建
- **重命名/删除** — 右键文件操作
- **IndexedDB 持久化** — 关闭后重新打开自动恢复之前的文件夹

### 文件预览
| 类型 | 说明 |
|------|------|
| Markdown | 渲染预览 + 编辑 + 保存 |
| HTML | 实时预览（外部修改自动刷新）+ 编辑 + 保存 |
| PDF | 浏览器原生阅读器 |
| 图片 | 直接显示 |
| 文本/代码 | 查看 + 编辑 + 保存 |
| Office | 显示文件信息，引导用系统默认程序打开 |

### HTML 实时预览
打开 HTML 文件后切换至预览模式，在 VS Code 等编辑器中修改保存，预览自动刷新（1.5 秒检测间隔）。

### 快捷键
| 快捷键 | 功能 |
|--------|------|
| `⌘P` | 快速搜索文件 |
| `⌘S` | 保存当前编辑的文件 |
| `⌘W` | 关闭当前标签 |
| `⌘⇧T` | 恢复关闭的标签 |

## 技术实现

- 纯静态页面，零依赖，零构建
- 基于浏览器 File System Access API 读写本地文件
- IndexedDB 持久化文件夹配置
- Apple 风格 UI（毛玻璃侧边栏、SF 字体、浅色主题）

## 浏览器兼容

需要 Chrome / Edge 108+ 或支持 File System Access API 的浏览器。Safari 暂不支持。
